HumanSense is a human-centric movie recommendation system that adapts
recommendations based on user purpose, identity, emotional tolerance,
and novelty preference.

## Key Features
- Purpose-driven recommendations (RELAX, UPLIFT, EXPLORE, DISTRACT)
- Semantic understanding using embeddings
- Custom HumanSense scoring framework
- Explainable and ethical recommendation logic
- Interactive UI using Streamlit

## Tech Stack
- Python
- Pandas, NumPy
- Sentence Transformers
- Streamlit

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the UI:
   streamlit run src/ui_app.py
